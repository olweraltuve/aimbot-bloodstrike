🚀 PASOS PARA INICIAR:
───────────────────────────────────────────────────────────────────

1️⃣  DESCARGAR EL MODELO (Primera vez)
   ▶ Ejecuta: download_yolov8_world.bat
   
   O se descargará automáticamente al iniciar el program_t.

2️⃣  INICIAR EL PROGRAM_T
   ▶ Ejecuta: start.bat
   
   El modelo usa estas clases para detectar humanoides:
   • "person"    - Personas en general
   • "human"     - Seres humanos
   • "player"    - Jugadores en videojuegos
   • "character" - Personajes y avatares

3️⃣  VERIFICAR FUNCIONAMIENTO
   ✓ No debe aparecer texto sobre las detecciones
   ✓ Solo verás cajas de colores:
     - Verde = Objetivo locked
     - Azul = Apuntando al objetivo
   ✓ Punto de mira en la cabeza calculada
   ✓ Línea amarilla del objetivo al centro
   ✓ Cruz roja en el centro de la pantalla

⚙️  PERSONALIZACIÓN (OPCIONAL):
───────────────────────────────────────────────────────────────────

Para cambiar las clases detectadas:

📁 Edita: lib/core/program_t_engine.py (línea ~146)

   Cambia:
   self.model.set_classes(["person", "human", "player", "character"])

   Por ejemplo:
   
   🎮 Para FPS:
   self.model.set_classes(["soldier", "enemy", "player", "combatant"])
   
   🧟 Para juegos de zombies:
   self.model.set_classes(["zombie", "infected", "monster", "enemy"])
   
   🏃 Para Battle Royale:
   self.model.set_classes(["player", "character", "person", "enemy"])

🐛 SOLUCIÓN DE PROBLEMAS:
───────────────────────────────────────────────────────────────────

❌ Error: "Model not found"
   💡 Solución:
   cd lib
   python -c "from ultralytics import YOLO; YOLO('yoloe-11l-seg.pt')"

❌ Error: "set_classes not found"
   💡 Solución:
   pip install --upgrade ultralytics

❌ No detecta nada
   💡 Solución:
   • Reduce 'confidence' a 0.35 en lib/config/game_profiles.json
   • Prueba diferentes clases de detección
   • Verifica que el juego esté en modo Ventana sin bordes

❌ FPS muy bajo
   💡 Solución:
   • Ejecuta setup_cuda.bat para configurar GPU
   • Verifica CUDA: python -c "import torch; print(torch.cuda.is_available())"
   • Reduce FOV en la configuración
